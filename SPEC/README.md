# README SPEC MACROS REPEATER

Проект отличается от остальных:

- Пока что для личного пользования, поэтому много упрощений
- Сборка
   - Нет "прод сборки", нет zip, нет dist, нет публикаций
   - Только Dev
- Менее строгие правила UI/UX
- Более строгие правила кода:
   - Максимальная простота и легковесность
   - Не используем lib/
   - Ничего не внедряем в DOM страниц, не перехватываем ничего про страницу сверх необходимого. Наша цель -- максимально имитировать человека и не вызывать подозрений у сайта, что это бот
