# PAGE ABOUT

- Header "MACROS REPEATER"
- (mouse-pointer-click) Records clicks on a page
- (play) Repeats recorded clicks
- (crosshair) Position mode: clicks recorded coordinates
- (code) Element mode: clicks page elements
- (eye) Visible mode: shows movement and clicks
- (eye-off) Stealth mode: works without visualisation
- (repeat-2) Repeats a macros up to 999 times
- (gauge) Execution speed: 0.25×, 0.5×, 1×, 2×
- (star) "Default" macros can be launched with a shortcut
- (shield-check) Doesn't use the network
- (git-fork) Code on GitHub // [https://github.com/md2it/browser-extension-macros-repeater](https://github.com/md2it/browser-extension-macros-repeater)
